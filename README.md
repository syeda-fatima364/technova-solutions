## 📌 Project Overview

This project consists of two main components:

- Part A: A C++-based Salary Management System  
- Part B: A professional company landing page created using Google Sites  

Together, these demonstrate both technical programming skills and basic web design/presentation skills applied to a real-world business scenario.

---

## 🖥️ Part A: C++ Salary Management Program

### 🚀 Description
This program automates the salary calculation process for TechNova Solutions, replacing manual spreadsheet-based methods. It takes employee data as input, performs salary computations, and generates detailed reports.

---

### ✨ Features

- Stores multiple employee records:
  - Name  
  - Employee ID  
  - Basic Salary  
  - Performance Rating  

- Input Validation:
  - Salary cannot be negative  
  - Performance rating must be A, B, C, or D  

- Salary Calculations:
  - Allowance:
    - 20% (Below Rs. 30,000)  
    - 15% (Rs. 30,000 – 59,999)  
    - 10% (Rs. 60,000+)  

  - Tax:
    - 0% (Below Rs. 40,000)  
    - 5% (Rs. 40,000 – 69,999)  
    - 10% (Rs. 70,000+)  

  - Performance Bonus:
    - A → Rs. 10,000  
    - B → Rs. 5,000  
    - C → Rs. 2,000  
    - D → Rs. 0  

- Outputs:
  - Individual salary reports  
  - Summary report including:
    - Total net salary  
    - Average net salary  
    - Highest net salary  

---

### 🧠 Programming Concepts Used

- Arrays  
- Loops  
- Functions  
- Conditional Statements  
- Input Validation  

---

### 📐 Salary Formulas

Allowance = Basic Salary × Allowance Rate  
Gross Salary = Basic Salary + Allowance  
Tax = Gross Salary × Tax Rate  
Net Salary = Gross Salary - Tax + Bonus  

---

### ▶️ How to Run

1. Compile the program:
   g++ salary.cpp -o salary

2. Run the program:
   ./salary
